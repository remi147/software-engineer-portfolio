## 🏃 Scrum & Agile Methodology

All projects in this portfolio are developed following **Scrum framework** principles:

```
📌 Product Backlog  →  🗓 Sprint Planning  →  🏃 2-Week Sprint
        ↓                                            ↓
📋 Daily Standup  ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
        ↓
✅ Sprint Review  →  🔄 Sprint Retrospective  →  Next Sprint
```

### Scrum Roles I Practice
- **Developer** — writing user stories, implementing features, writing tests
- **Product Owner** (solo projects) — backlog grooming, prioritisation, acceptance criteria

### Agile Ceremonies Applied
| Ceremony | Frequency | Artifact |
|---|---|---|
| Sprint Planning | Every 2 weeks | Sprint Backlog |
| Daily Standup | Daily (async via commit notes) | Commit history |
| Sprint Review | End of sprint | Demo / release tag |
| Sprint Retrospective | End of sprint | `RETROSPECTIVE.md` |
| Backlog Grooming | Weekly | GitHub Issues |

See [`agile/`](./agile/) for sprint logs, retrospectives, and backlog templates.
